# 御製序

<a id="KR3k0059_WYG_000-1a"></a><span style="margin-left: 1em">欽定四庫全書</span>  
御製佩文韻府序  
<span style="margin-left: 1em">朕萬幾在御日昃宵分未遑自逸時當燕閒不輟問學</span>  
<span style="margin-left: 1em">羣經子史誦其文而晰其義矣以至百家之書凡可以</span>  
<span style="margin-left: 1em">禆世敎勵民風者脩眀補正㒺使闕遺嘗謂韻府羣玉</span>  
<span style="margin-left: 1em">五車韻瑞諸書事繫於字字統於韻稽古者近而取之</span>  
<span style="margin-left: 1em">約而能博是書之作誠不為無所見也然其為書簡而</span>  
<span style="margin-left: 1em">不詳略而不偹且引據多誤朕毎致意焉欲博稽衆籍</span>  
<a id="KR3k0059_WYG_000-1b"></a><span style="margin-left: 1em">著為全書爰於康熈四十三年夏六月朕與内直翰林</span>  
<span style="margin-left: 1em">諸臣親加考訂證其訛舛增其脫漏或有某經某史所</span>  
<span style="margin-left: 1em">載某字某事未脩者朕復時時面諭一一增録漸次成</span>  
<span style="margin-left: 1em">帙猶以故實或未極博於十月復命閣部大臣更加蒐</span>  
<span style="margin-left: 1em">采以裒益之既有原本增本又有内増外増將付剞劂</span>  
<span style="margin-left: 1em">矣名曰佩文韻府随於十二月開局武英殿集翰林諸</span>  
<span style="margin-left: 1em">臣合併詳勘逐日進覧旋授梓人於五十年十月全書</span>  
<span style="margin-left: 1em">告成共一百零六卷一萬八千餘頁嚢括古今網羅鉅</span>  
<a id="KR3k0059_WYG_000-2a"></a>細韻學之盛未有過於此書者也書成諸臣請序朕念  
自初至今經八年矣歴寒暑之乆積嵗月之勤朕於此  
書政事之暇未嘗惜一日之勞也朕又嘗諭諸臣從來  
著一大書非數十年之功不能成今数年以來所成大  
部書凡十有餘種若非合衆人之力豈能刻期告竣故  
念先後預事諸臣皆命列名其中兹序佩文韻府因備  
記編撰之始末遂及脩集諸書之大指以見成書之不  
易如此  
<a id="KR3k0059_WYG_000-2b"></a>康熈五十年十月題


# 提要

<a id="KR3k0059_WYG_000-3a"></a><span style="margin-left: 1em">欽定四庫全書　　　　　子部十一</span>  
御定佩文韻府　　　　　　　類書類  
<span style="margin-left: 3em">提要</span>  
<span style="margin-left: 5em"><sub>臣/</sub>等謹案</span>  
<span style="margin-left: 2em">御定佩文韻府四百四十四卷康熙五十年</span>  
<span style="margin-left: 2em">聖祖仁皇帝御定考唐書藝文志載顔真卿韻海鏡</span>  
<span style="margin-left: 5em">原二百卷釋皎然陪顔使君修韻海畢東溪</span>  
<span style="margin-left: 5em">泛舟餞諸文士詩有外史刋新韻中郎定古</span>  
<a id="KR3k0059_WYG_000-3b"></a><span style="margin-left: 5em">文菁華兼百氏縑雅備三墳句其自注又有</span>  
<span style="margin-left: 5em">魯公著書依切韻起東字脚語然則分韻𨽻</span>  
<span style="margin-left: 5em">事始自真卿今其書不傳宋元間作者頗夥</span>  
<span style="margin-left: 5em">謂之詩韻<sub>語詳韻府/羣玉條下</sub>其傳於今者惟韻府羣玉</span>  
<span style="margin-left: 5em">為最古至明又有五車韻瑞然皆疎漏不完</span>  
<span style="margin-left: 5em">舛訛相踵楊慎作韻藻朱彞尊作韻粹其子</span>  
<span style="margin-left: 5em">昆田又作三體摭遺皆欲補陰氏凌氏之闕</span>  
<span style="margin-left: 5em">而仍未賅備是以我</span>  
<a id="KR3k0059_WYG_000-4a"></a><span style="margin-left: 1em">聖祖仁皇帝特詔儒臣蒐羅典籍輯為是編毎字皆</span>  
<span style="margin-left: 4em">先標音訓所𨽻之事凡陰氏凌氏書所已採</span>  
<span style="margin-left: 4em">者謂之韻藻列於前兩家所未採者别標增</span>  
<span style="margin-left: 4em">字列於後皆以兩字三字四字相從而又各</span>  
<span style="margin-left: 4em">以經史子集為次其一語而諸書互見者則</span>  
<span style="margin-left: 4em">先引最初之書而其餘以次註於下又别以</span>  
<span style="margin-left: 4em">事對摘句附於其末原本不標卷第但依韻</span>  
<span style="margin-left: 4em">釐為一百六卷而中分子卷二十有四今以</span>  
<a id="KR3k0059_WYG_000-4b"></a><span style="margin-left: 4em">篇頁繁重編為四百四十四卷自有韻府以</span>  
<span style="margin-left: 4em">來無更浩博於是者俯視隂氏凌氏之書如</span>  
<span style="margin-left: 4em">滄海之于蠡勺矣考康熙五十九年大學士</span>  
<span style="margin-left: 4em">王掞等恭製韻府拾遺序有曰佩文韻府書</span>  
<span style="margin-left: 4em">成卷帙一百有六聞諸臣分纂之時毎繕初</span>  
<span style="margin-left: 4em">稿先呈</span>  
<span style="margin-left: 1em">御覽我</span>  
<span style="margin-left: 1em">皇上十行並下㸃摘闕遺舉凡六經奥義箋詁之所</span>  
<a id="KR3k0059_WYG_000-5a"></a><span style="margin-left: 4em">難通四部僻書棗梨之所未鋟莫不</span>  
<span style="margin-left: 1em">親加批乙宣付諸臣再三稽考雖諸臣衆手合作之</span>  
<span style="margin-left: 4em">書實我</span>  
<span style="margin-left: 1em">皇上一心裁定之書也云云盖由</span>  
<span style="margin-left: 1em">聖學髙深為千古帝王所未有故是書博贍亦千古</span>  
<span style="margin-left: 4em">著述所未有也乾隆四十五年十二月恭校</span>  
<span style="margin-left: 4em">上</span>  
<span style="margin-left: 7em">總纂官<sub>臣/</sub>紀昀<sub>臣/</sub>陸錫熊<sub>臣/</sub>孫士毅</span>
